# 人脸分类模块说明

本目录公开的是 **模型结构代码与推理代码**，用于说明本项目中“第一阶段人脸筛选”的实现方式。

## 包含文件

- `face_classifier.py`：训练脚本，定义了模型结构、数据集加载、训练流程、评估流程
- `prediction.py`：推理脚本，用于对图片目录进行批量筛选，保留人脸图片

## 不包含内容

出于体积与模型资产管理考虑，本发布版 **不包含训练权重**：

- `face_classifier.pth`

如果你希望完整运行推理流程，请自行准备训练好的权重文件，并放到：

```text
classification/face_classifier.pth
```

## 当前模型结构

项目当前使用：

- `EfficientNet-B0`

并替换了末端分类头用于做人脸 / 非人脸二分类。

## 用途

该目录主要用于：

1. 公开模型结构代码
2. 说明项目中“人脸筛选”阶段的具体实现
3. 方便后续二次训练或替换模型

## 注意

- 训练脚本中的数据集路径为占位示例，需要你自行修改
- 推理脚本中的权重文件路径也需要与你本地文件保持一致
